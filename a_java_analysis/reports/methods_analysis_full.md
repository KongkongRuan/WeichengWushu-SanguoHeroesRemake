## 二、J2ME 回调方法

### 2.1 run() — 主游戏循环 (行23476)

```
原始签名: public final void run()
语义名: gameLoop
```

**功能**: 游戏主线程入口,实现 10 FPS 帧率控制的状态机循环。

**流程**:
1. 检查 `B1171`(线程标志),若已有线程运行则调用 `c()` 等待并设置 `bV` 状态
2. 设置 `B1171=1`,创建同步锁 `a1172`
3. 进入 `while(true)` 主循环:
   - 检查 `b1018`(运行标志),false 则调用 `CMidlet.a()` 退出
   - 记录帧开始时间
   - 调用 `G()` — 主帧更新(状态机分发)
   - 清除按键状态 `h1047=0`
   - `repaint()` + `serviceRepaints()` 触发重绘
   - `Thread.sleep(10)` 基础延时
   - `Thread.yield()` 让出 CPU
   - 等待达到 100ms 帧间隔(目标 10 FPS)
   - 若音效开启,统计帧耗时到 `s`
   - 帧计数器 `a1019 += 1`
