### 模块 06: a-z 逻辑方法 ✅ (含本轮修复)

#### ★ `a(int,int,int)` 角度计算 — 本轮修复

**VF源码** (L10009-10072):
```java
var7 = var8;  // var7 = 45 (预设mid值)
for (int var18 = var10; var18 > var12; var7 = var18 + var12 >> 1) {
    int var15 = this.j1049[var7];  // 使用var7(首次=45)
    // ... 二分查找逻辑
}
```

**问题**: VF的for循环更新表达式 `var7 = var18 + var12 >> 1` 在循环体**之后**执行。首次迭代使用预设值 `var7 = 45`。但之前代码在循环体**开头**计算 `mid = (hi + lo) >> 1 = (89 + 0) >> 1 = 44`，导致首次检查的索引错误(44而非45)。

**修复前** (错误):
```java
while (hi > lo) {
    mid = (hi + lo) >> 1;  // ★ 首次得到44, 应为45!
    int val = this.j1049[mid];
    ...
}
```

**修复后** (正确):
```java
int mid = 45;  // ★ 预设值, 同 VF var7 = var8 = 45
while (hi > lo) {
    int val = this.j1049[mid];  // 首次使用预设值45
    if (val > absN) {
        hi = mid - 1;
    } else {
        if (val >= absN) break;
        lo = mid + 1;
    }
    if (hi <= lo) {
        mid = lo;
        break;
    }
    mid = (hi + lo) >> 1;  // ★ 在末尾更新, 供下次迭代使用
}
```

**影响**: 此bug会导致角度计算首次比较检查 `j1049[44]` 而非 `j1049[45]`，可能导致部分角度计算偏差1度，进而影响敌人移动方向和塔攻击角度。已修复 ✓
