## 三方反编译器对比

| 反编译器 | 成功率 | 特点 |
|----------|--------|------|
| Vineflower | ~90% | 主源，部分方法失败(run/d/b/av/aw) |
| CFR | ~85% | 部分方法失败(d/A等)，输出格式有时混乱 |
| Procyon | ~88% | 部分方法失败(d等)，但**保留原始字节码**，极其有价值 |

**Procyon的独特价值**: 当反编译失败时，Procyon会输出完整的原始字节码注释，这使我们能够精确恢复 `d()` 方法的完整逻辑。
