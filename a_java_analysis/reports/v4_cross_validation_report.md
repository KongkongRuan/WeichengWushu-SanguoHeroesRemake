## 详细问题与修正

### 模块 02: `02_constructor.java` — c1051 数据严重错误 ❌

#### 问题: c1051 (塔建造路径) 数组数据完全错误

**VF 正确数据** (L1224-1245):
```java
byte[][] var35 = new byte[4][];  // 只有4行!
var35[0] = {0, 1, 2, 3, 4};      // 5个元素 (主路径)
var35[1] = {1, 2, 3, 4};          // 4个元素
var35[2] = {6, 7, 8, 9};          // 4个元素
var35[3] = {5, 7, 8, 9};          // 4个元素
// 不存在第5行!
```

**模块02错误数据** (修正前):
```java
byte[][] var20 = new byte[][] {
    {1, 2, 3, 4},      // 错! 应为 {0, 1, 2, 3, 4}
    {1, 2, 3, 4},
    {6, 7, 8, 9},
    {5, 7, 8, 9},
    {0, 0, 0, 0}        // 错! 不存在此行
};
```

**错误明细**:
1. 多了第5行 `{0, 0, 0, 0}` (VF中不存在)
2. 第1行应为 `{0, 1, 2, 3, 4}` (5个元素), 错误地写成了 `{1, 2, 3, 4}` (4个元素)
3. 数组维度应为 `byte[4][]`, 错误地写成了 `byte[5][]`

**影响**: 塔建造路径数据完全错误, 会导致游戏建造系统逻辑异常

**修正**: ✅ 已按 VF L1224-1245 修正为正确的4行数据
